# Kubeflow Pipelines Roadmap

## Overview

The Kubeflow Pipelines project roadmap is tracked using [GitHub Milestones](https://github.com/kubeflow/pipelines/milestones), following the Kubeflow Release Team's request: this approach has been recommended in order to maintain a detailed view of all the items intended for upcoming Kubeflow releases, organized by "themes". The roadmap is continuously updated as we make progress towards a Kubeflow Release.

## Current Roadmap

The current roadmap and upcoming features are tracked in our [GitHub Milestones](https://github.com/kubeflow/pipelines/milestones). Each milestone tracks a specific or future Kubeflow release and contains the planned features, enhancements, and bug fixes.

For a full list of past milestones, see [closed milestones](https://github.com/kubeflow/pipelines/milestones?state=closed).
