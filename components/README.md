# Components Migration

Most legacy contents of this directory have been removed from `kubeflow/pipelines`.

Community-maintained components now live in the
[`kubeflow/pipelines-components`](https://github.com/kubeflow/pipelines-components)
repository.

The Google-maintained components under `components/google-cloud` are the current
exception and are still being migrated to the separate repository.

For migration details and background, see the original announcement:
[`The Components directory will be moved a separate Repository`](https://github.com/kubeflow/pipelines/discussions/12218),
followed by the later reminder:
[`Reminder: @components/ will be removed on May 11, 2026`](https://github.com/kubeflow/pipelines/discussions/13051).
